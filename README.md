# SaaS Landing Page Template

Sprint 1 foundation for a polished B2B SaaS marketing site template.

## Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui-style source components
- lucide-react

## Sprint 1 Scope

- Vite + React + TypeScript app scaffold
- Tailwind and shared design tokens
- shadcn-compatible configuration and UI primitives
- Layout primitives: `Container`, `Section`, `SectionHeader`
- Responsive navigation and footer shells
- Placeholder page composition for future landing sections

## Scripts

- `npm run dev`
- `npm run build`
- `npm run lint`

## Source Structure

```text
src/
  components/
    layout/

    ui/
  lib/
```
